Local Database transformation into OMOP-CDM

HM Hospitales

This page provides links to various data profiling reports from HM Hospitales' local database. The main focus of this compilation of data profiling reports is to have a broad understanding of the nature of the data we are dealing with and that is subject to transformation into standard concepts using the OMOP-CDM guidelines. These reports includes various pieces of information such as dataset statistics, distribution of values, missing values, memory usage, etc., which are very useful for exploring and analyzing data efficiently. Click on a link to view the data profile for the respective table or combination of tables.
